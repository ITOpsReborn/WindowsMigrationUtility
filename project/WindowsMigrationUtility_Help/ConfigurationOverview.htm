WMU Configuration User Interface Overview

The WMU application utilizes user settings stored in the user's %APPDATA% folder, i.e. C:\Users\YOURNAME\AppData\Local. The WMU Configuration Form reads and writes to this XML file.



USMT Configuration File
The general configuration for ScanState and/or LoadState utility.
USMT Custom Files
This is where you name your custom capture / restore files for USMT, i.e. "MigApp.xml", "MigUser.xml".
Capture - Before Process Command
Many times in organizations a command needs to run before the ScanState.exe is launched so certain settings are captured and moved to the new OS. Place the script inside the ".\Source\Config\". This will make sure this script gets copied to any remote system you need this to be ran on.
Restore - After Process Command
Many times in organizations a command needs to be ran after the LoadState.exe is completed. Place the script inside the ".\Source\Config" folder. This will make sure this script gets copied to any remote system you need this to be ran on.
